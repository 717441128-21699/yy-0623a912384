## 1. 产品概述

材料进场验收系统是一款面向大型房建项目总包项目部的 Web 应用，旨在将传统纸质验收单转化为清晰可追溯的电子台账。系统覆盖材料员录入到场信息、质检员现场核验、监理工程师线上签署的完整流程，实现材料进场验收的全数字化闭环管理。

- 解决传统纸质验收单易丢失、难追溯、签字滞后的问题
- 目标用户：总包项目部材料员、质检员、监理工程师

## 2. 核心功能

### 2.1 用户角色

| 角色 | 进入方式 | 核心权限 |
|------|----------|----------|
| 材料员 | 角色切换选择 | 录入到场批次信息、上传送货单照片、补传整改结果 |
| 质检员 | 角色切换选择 | 逐项勾选检查项、给出验收结论（可接收/需复检/拒收） |
| 监理工程师 | 角色切换选择 | 查看验收单详情与照片、线上签署意见、生成整改单 |

### 2.2 功能模块

1. **待验收批次**：材料员按类别录入供应商、规格型号、合同数量、车牌、到场时间，上传送货单照片；质检员查看待验清单
2. **现场验收单**：质检员逐项勾选检查项（外观、数量、合格证、检测报告），系统自动给出验收状态；监理查看验收单并签署意见
3. **问题闭环**：监理/质检员生成整改单，指定责任单位与复查日期；材料员补传处理结果后关闭

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 待验收批次 | 批次录入表单 | 选择材料类别（钢筋/水泥/防水卷材等），填写供应商、规格型号、合同数量、车牌号、到场时间，上传送货单照片 |
| 待验收批次 | 批次列表 | 按状态（待验收/验收中/已完成）筛选展示所有到场批次，支持搜索和分类筛选 |
| 待验收批次 | 批次详情抽屉 | 查看批次完整信息，包含送货单照片预览 |
| 现场验收单 | 验收检查表 | 逐项勾选：外观质量、数量核对、合格证、检测报告，每项可选符合/不符合 |
| 现场验收单 | 自动状态判定 | 根据检查项结果即时给出"可接收/需复检/拒收"状态 |
| 现场验收单 | 监理签署区 | 监理查看照片与检查项结果，填写签署意见，确认签署 |
| 现场验收单 | 验收单列表 | 展示所有验收单，按状态和日期筛选 |
| 问题闭环 | 整改单生成 | 从验收单中发起整改单，填写问题描述、指定责任单位、复查日期 |
| 问题闭环 | 整改单列表 | 按状态（待整改/整改中/已关闭）展示所有整改单 |
| 问题闭环 | 结果补传与关闭 | 材料员上传整改处理结果，监理确认后关闭整改单 |

## 3. 核心流程

**材料到场 → 材料员录入批次 → 质检员现场验收 → 系统自动判定状态 → 监理签署 → 如有问题生成整改单 → 材料员整改 → 监理确认关闭**

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：建筑工业风格——深蓝灰（#1B2A4A）为主色，安全橙（#E8652A）为强调色，代表工地警示与专业感
- **辅助色**：浅灰底（#F5F6FA）、白色卡片、绿色（#2D8C4E）表示通过、红色（#D64545）表示拒收、琥珀色（#D4A017）表示复检
- **按钮风格**：圆角矩形（8px），主按钮填充强调色，次按钮描边，危险操作红色
- **字体**：标题使用 Noto Sans SC Bold，正文使用 Noto Sans SC Regular
- **布局风格**：左侧固定导航栏 + 右侧内容区，卡片式内容展示，表格采用斑马纹
- **图标风格**：线性图标（Lucide Icons），与工业简洁风格匹配

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 待验收批次 | 批次录入表单 | 模态弹窗，分步表单（类别选择→基本信息→照片上传），下拉选择+文本输入+日期选择器+图片上传 |
| 待验收批次 | 批次列表 | 筛选栏（状态标签+类别下拉+搜索框）+ 数据表格（斑马纹行）+ 状态徽标 |
| 待验收批次 | 批次详情 | 右侧抽屉滑出，信息卡片 + 照片网格预览 |
| 现场验收单 | 验收检查表 | 卡片式检查项列表，每项左右布局（检查名称+勾选切换），底部状态判定区 |
| 现场验收单 | 监理签署区 | 文本域输入意见 + 电子签名确认按钮，签署后显示签名时间戳 |
| 现场验收单 | 验收单列表 | 三栏看板视图（可接收/需复检/拒收），卡片拖拽感布局 |
| 问题闭环 | 整改单生成 | 模态弹窗，表单填写问题描述+责任单位下拉+日期选择 |
| 问题闭环 | 整改单列表 | 时间线式布局，状态标签颜色区分，支持筛选 |
| 问题闭环 | 结果补传 | 抽屉式面板，图片上传区+文字说明+提交按钮 |

### 4.3 响应式设计

- 桌面优先设计，最小支持 1280px 宽度
- 平板端（768px-1024px）：导航栏折叠为图标模式，表格横向滚动
- 移动端（<768px）：底部标签导航，卡片单列堆叠，表单全屏展示

### 4.4 设计特色

- 顶部显示项目名称与当前角色切换器，模拟真实工地场景
- 状态流转采用颜色编码，一目了然
- 照片预览支持灯箱放大查看，方便核验细节
- 整改单时间线展示完整处理过程，确保可追溯
